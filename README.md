# Westworld Saloon Piano Simulator V2

This version uses the detailed generated Wurlitzer-style piano image as the visual base, so the program looks much closer to the requested reference.

## Files

- `index.html`
- `style.css`
- `script.js`
- `assets/piano-main.png`

## GitHub Pages

1. Create a new GitHub repository.
2. Upload everything from this ZIP.
3. Go to `Settings → Pages`.
4. Select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.

## Notes

The sound is generated in the browser with demo note patterns.
No copyrighted Westworld MIDI/audio files are included.
