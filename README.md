# Westworld Saloon Piano Simulator

Static GitHub Pages web app with:

- saloon / Wurlitzer-style player piano UI
- animated paper roll
- animated keys and hammers
- playlist
- synthesized browser audio
- responsive layout

## Upload to GitHub

1. Create a new GitHub repository.
2. Upload all files from this ZIP.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, select:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. Open your GitHub Pages URL.

## Important

The included songs are browser-generated demo patterns, not original copyrighted Westworld audio.  
You can later replace or extend the song system with real MIDI parsing.
