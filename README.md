# Westworld Saloon Piano Simulator V3

Diese Version ist neu aufgebaut und benutzt kein großes Screenshot-Overlay mehr als Programmoberfläche.

## Inhalt

- echtes HTML/CSS Interface
- echtes CSS/SVG-artiges Klavier
- animierte Papierrolle
- animierte Mechanik/Hämmer
- animierte Tasten
- Browser-Sound
- GitHub Pages kompatibel

## Installation auf GitHub

1. Neues GitHub Repository erstellen
2. Alle Dateien aus dieser ZIP hochladen
3. Settings → Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. Speichern
